# ✅ Google فصحى سينمائية - Preset النهائي

## نظرة عامة

تم تطبيق **الخيار (A) - Google فصحى سينمائية** في تطبيق سامكو لصناعة المحتوى بالذكاء الاصطناعي.

## المميزات

✅ نطق عربي صحيح كما هو مكتوب
✅ ممتاز للإعلانات، الريلز، المقاطع السينمائية
✅ بدون لهجة محلية (فصحى واضحة)
✅ مناسب لكل الدول العربية

## الإعداد المختار

| المعيار | القيمة |
|---------|--------|
| **اللغة** | العربية الفصحى (ar-XA) |
| **الصوت** | ar-XA-Chirp3-HD-Kore (صوت سينمائي) |
| **الأسلوب** | سينمائي إعلاني واضح |
| **الضمان** | قراءة حرفية للنص بدون إعادة صياغة |

## Preset النهائي

### voiceName
```
ar-XA-Chirp3-HD-Kore
```

### styleHint / Prompt
```
أنت قارئ صوتي محترف.
اقرأ النص العربي التالي كما هو مكتوب حرفيًا دون أي تعديل أو إعادة صياغة.
نطق عربي فصيح واضح جدًا، مخارج حروف دقيقة، التزم بعلامات الترقيم.
بنبرة سينمائية فخمة، سرعة متوسطة، وقفات طبيعية بعد الجمل.
لا تغيّر الكلمات، لا تفسّر المعنى، اقرأ النص فقط.
```

## متى تستخدمه؟

- ✅ **إعلانات** - نبرة احترافية فخمة
- ✅ **فيديوهات TikTok / Reels** - محتوى جذاب
- ✅ **تعليق سينمائي احترافي** - صوت مميز
- ✅ **محتوى عام لكل الدول العربية** - فصحى مفهومة

## كيفية الاستخدام

### 1. Frontend (تلقائي)
التطبيق يستخدم هذا الـ Preset افتراضياً عند اختيار:
- اللغة: عربي (أي لهجة)
- الصوت: كوري (أنثى) أو أي صوت عربي
- الأسلوب: سينمائي (أو افتراضي)

### 2. Backend API
```bash
curl -X POST http://localhost:8080/tts \
  -H "Content-Type: application/json" \
  -d '{
    "text": "مرحبا بكم في عالم سامكو لصناعة المحتوى بالذكاء الاصطناعي",
    "voiceName": "ar-XA-Chirp3-HD-Kore",
    "returnWavBase64": true
  }'
```

**ملاحظة**: لا حاجة لإرسال `styleHint` - سيستخدم Backend الـ Preset الافتراضي تلقائياً.

### 3. مع styleHint مخصص
```bash
curl -X POST http://localhost:8080/tts \
  -H "Content-Type: application/json" \
  -d '{
    "text": "مرحبا بكم",
    "voiceName": "ar-XA-Chirp3-HD-Kore",
    "styleHint": "أنت قارئ صوتي محترف...",
    "returnWavBase64": true
  }'
```

## الفرق بين الأساليب

### السينمائي (الافتراضي)
```
بنبرة سينمائية فخمة، سرعة متوسطة، وقفات طبيعية بعد الجمل.
```
**الاستخدام**: إعلانات، ريلز، محتوى احترافي

### الهادئ
```
بنبرة هادئة مطمئنة، سرعة بطيئة قليلاً، دون مبالغة.
```
**الاستخدام**: قصص، محتوى مريح

### الحماسي
```
بنبرة حماسية سريعة قليلاً، مع وضوح شديد.
```
**الاستخدام**: أخبار، رياضة، محتوى حماسي

### الطفولي
```
بصوت طفولي لطيف مع الحفاظ على نطق العربية الصحيح.
```
**الاستخدام**: قصص أطفال، محتوى تعليمي

```typescript
function getStyleHint(locale: string, speed?: string, pitch?: string, style?: string): string {
  const isArabic = locale.startsWith('ar-');
  
  if (!isArabic) {

  }

  // البرومبت الأساسي الصارم للعربية (فصحى سينمائية)

لا تغيّر الكلمات، لا تفسّر المعنى، اقرأ النص فقط.`;

  // إذا لم يتم تحديد أسلوب، استخدم السينمائي كافتراضي
  if (!style || style === 'cinematic') {
    return ARABIC_CINEMATIC_BASE;
  }
  
  // ... أساليب أخرى
}
```
